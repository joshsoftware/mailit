Make My Event

Organization of events like blood donation, running, cycling, tech meetup in their area is made simple through this portal.

Anyone who wants to organize any such event can register on this site & then he/she can create the event and Publish it. Users can browse, share, participate for the events. Organizer can manage event, display participants, upload photos, track event throughout.

KuchBhi

KuchBhi is a simple classified ads platform. Using this portal user can register through social networking sites like Facbook, Twitter and post ads and search ads by location, categories and tags.

Dynamic roles and permissions using cancan

Most of us already use cancan for authorization, where we define the Role Based Access (RBAC) to various models in the Ability class. However, any changes to these abilities requires code changes to the Ability class and restarting the application for the changes to take effect. What if we could assign these permissions dynamically ? This is how it can be achieved.
