OpenSource Contrib

FeatherCMS - We have felt the pain in integrating simple content pages in a Rails portal. Any current CMS (gem or framework) makes this a heavy procedure - in memory or in effort. So, we built FeatherCMS - a simple lightweight Do-It-Yourself content manager. 


gem install feather_cms
rails generate feather_cms [NAME] [STORAGE] [options]
This generates a feather_cms route, controller and view. You can edit the pages using CodeMirror and see a simple preview!
