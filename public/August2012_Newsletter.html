Lazyload Images

Attractive images are the heart of any application. However the page loading time can drastically increase if the application takes huge amount of time to load these images! For such situations you need some way to reduce loading time for images. Lazyload Jquery Plugin would help you to achieve this. This is how you can set it up for your application.


Migrating to new Heroku Postgres plans

Recently Heroku released new version of postgres 9.1 with new price plans. So there comes a need to migrate from existing shared database to one of the new plans released with Postgres 9.1. As per this new plan , limit on using free shared database is changed from db size (5 MB) to limit on rows (10,000 rows). This post will guide you to accomplish this migration.
